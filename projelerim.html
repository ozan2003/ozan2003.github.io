Projelerim

  • Berber Yönetim Sistemi - Berber işletmeleri için web tabanlı yönetim sistemi. ASP.NET Core ve Entity Framework Core kullanılarak kullanıcı kaydı ve girişi, randevu yönetimi ve çalışan profilleri gibi fonksiyonları destekler.
  • Heartbeat Monitor - Ham ICMP Echo mesajlarını, sunucuların sistem verilerini toplamak ve doğrulamak için kullanan hafif bir Python istemci-sunucu izleme aracı. Telemetri verilerini SQLite ile saklar.
  • RFC Reader - IETF RFC belgelerini RFC Editor web sitesinden çekerek düzenli formatta görüntüleyen, Rust ile geliştirilmiş terminal tabanlı RFC görüntüleyici.
